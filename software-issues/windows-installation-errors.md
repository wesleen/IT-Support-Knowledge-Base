# Windows Installation Errors - Troubleshooting Guide

## Problem
Windows fails to install or shows errors during installation.

## Common Errors
- “Windows cannot be installed on this disk”
- Installation freezing
- Missing drivers
- Boot errors

## Possible Causes
- Corrupted installation media
- Wrong disk format (MBR vs GPT)
- Faulty hard drive
- BIOS misconfiguration

## Troubleshooting Steps
1. Restart installation process
2. Recreate bootable USB using official Windows Media Tool
3. Enter BIOS and check boot mode (UEFI/Legacy)
4. Format disk correctly (GPT recommended for UEFI)
5. Disconnect unnecessary drives

## Advanced Fixes
- Run disk check (chkdsk)
- Replace faulty storage drive
- Update BIOS firmware

## Resolution
Reinstall Windows using correct boot settings and clean installation media.
