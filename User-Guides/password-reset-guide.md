# Password Reset Guide

## Problem
User forgot password or cannot log in.

## Possible Causes
- Forgotten password
- Account locked due to failed attempts
- Expired password policy

## Steps to Reset Password

### For Windows Login:
1. Click “Forgot Password”
2. Follow security questions or email verification
3. Reset password and log in again

### For Microsoft Account:
1. Go to https://account.live.com/password/reset
2. Enter email or phone number
3. Verify identity via code
4. Create new password

## IT Admin Reset (If applicable)
- Use admin account
- Reset password via Control Panel or Active Directory

## Prevention Tips
- Use password manager
- Create strong but memorable passwords
- Update recovery email and phone number
